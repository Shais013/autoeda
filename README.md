## What it does

AutoEDA automates the most time-consuming part of any data science project — exploratory data analysis. Upload a CSV and get:

- **Dataset Overview** — shape, types, memory usage, duplicates
- **Missing Value Analysis** — heatmaps, per-column breakdown
- **Descriptive Statistics** — mean, std, skewness, kurtosis for all numeric columns
- **Distribution Plots** — histograms + boxplots for every numeric feature
- **Categorical Analysis** — top-N value counts per category
- **Correlation Heatmap** — with strong correlation detection
- **Outlier Detection** — IQR-based, with count and bounds per column
- **AI Insights** — plain English narrative generated by Claude AI

## Local Setup

```bash
# 1. Clone the repo
git clone https://github.com/yourusername/autoeda.git
cd autoeda

# 2. Create virtual environment
python -m venv venv
source venv/bin/activate  # Windows: venv\Scripts\activate

# 3. Install dependencies
pip install -r requirements.txt

# 4. Set your API key
echo "ANTHROPIC_API_KEY=your_key_here" > .env

# 5. Run the app
streamlit run app.py
```
